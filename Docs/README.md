## 💡 功能概览

MetaX是一个Python工具库，提供以下功能：

### 数学运算
- 基础运算：加、减、乘、除
- 高级运算：幂、开方、三角函数、对数等
- 数论函数：阶乘、最大公约数、最小公倍数

### 字符串处理
- 格式化工具：大小写转换、反转、截断、脱敏等
- 验证工具：邮箱、手机号、身份证、URL、IP等验证

### 日期时间
- 时间获取：当前时间、时间戳
- 时间计算：日期加减、时间差
- 日期判断：周末、工作日、年龄计算

### 文件操作
- 文件读写：文本、JSON、按行读写
- 文件信息：大小、扩展名、路径操作
- 文件操作：复制、移动、删除、哈希计算
